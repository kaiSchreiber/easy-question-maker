Easy Question Maker

Ein kleines Programm zum Zusammenstellen von Bilderrätseln. Die Fotos werden auf dieselbe Höhe gebracht und nebeneinander montiert.

A Installation
PC: eqm.exe runterlasen. Das Programm muss nicht installiert werden und ist direkt ausführbar.
Mac: eqm.dmg runterlasen. Image durch Doppelklick mounten. Das Programm ist direkt vom Image aus ausführbar, kann aber auch aufs Desktop oder in den Programm-Ordner verschoben werden.

B Bilder zufügen
PC: Bilder können per Drag and Drop als Datei oder direkt aus Firefox auf die Programmoberfläche gezogen werden und werden dem Rätsel dann rechts zugefügt.
Mac: Drag und Drop funktioniert nur für Dateien und nur über das Dock-Icon des Programmes (die beiden Fragezeichen im Dock).
Falls Drag/Drop nicht funktionieren sollte (getestet für OSX10.7 und Windows 7) können Bilder auch über den Lade-Knopf zugefügt werden.

C Bilder beschneiden:
Jedes Bild kann nach dem Zufügen angeklickt werden. Dann öffnet sich ein neues Fenster, in dem es beschnitten werden kann. Linker Mausklick setzt die linke obere Ecke, rechter Mausklick die rechte untere. Schließen des Fensters übernimmt die Änderungen. Wird beim Mausklick die Shift-Taste gehalten, wird die jeweilige Ecke entfernt (Rückkehr zum Vollbild).

D Bilder entfernen
Rechtsklick auf ein Bild in der Übersicht entfernt es aus dem Rätsel.

E Rätsel speichern
Speichern schreibt das gesamte Rätsel als Bilddatei in der vorgegebenen Höhe (Default 400 Pixel).
